Overview
------------
Time to Budget is an iOS application used for time management.

Downloading
-----------------
Time to Budget, when available, can be found on the Apple App Store.


Documentation
--------------------
Documentation can be found in the [repository wiki](https://bitbucket.org/arrkensoftware/timetobudget_ios/wiki/Home).

Development
------------------
Time to Budget development is hosted on bitbucket at this [repository](https://bitbucket.org/arrkensoftware/timetobudget_ios).

Bug Reporting
-------------------
You can report bugs using the bitbucket issue tracking tool found [here](https://bitbucket.org/arrkensoftware/timetobudget_ios/issues?status=new&status=open).
